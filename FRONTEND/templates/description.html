Főoldal -> célja: adatok hozzáadása, real-time statisztikák megtekintése

Bevétel / Kiadás hozzáadása
  • Bevétel / Kiadás típúsának megadása
  • Összeg meghatározása
  • Statisztika gyors hozzáadása egy kattintással
  • A valuta típusa is szabadon megadható és elmenthető
Túlköltekezés kezelése
  • Túlköltekezés esetén a megtakarítás vastag piros színnel fog megjelenni
  • Amíg Túlköltekezésben vagyunk, minden hozzáadott adat után alert üzenetet kapunk a túlköltekezés mértékéről
Statisztikák vizuális megjelenítése
  • Egyszerű megjelenítés switch bekapcsolásával
  • Minden hozzáadott adat megjelenítésre kerül, frissített statisztikával
  • Százalékos megjelenítés -> az adott sor hány százalékát teszi ki az összesített bevételnek / kiadásnak?
  • Külön összesített táblázat a legnépszerűbb költeketésekről

Szűrt statisztika oldal -> célja: Bevételek / Költekezések kategorikus eloszlásának vizuális ábázolása

Kategorikus statisztikák megtekintése
  • Minden kategória egyszer jelenik meg a táblázatokban
  • Az adott kategóriákba tartozó minden adat szummázódik
  • Statisztika gyors hozzáadása egy kattintással
